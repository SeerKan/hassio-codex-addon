# Codex Agent

Ask Codex to inspect, propose, and apply Home Assistant changes from the
sidebar. This panel is available only to Home Assistant administrators.

Each administrator signs in to Codex separately, so usage is charged to or
limited by that user's own Codex entitlement.

The add-on also ships Home Assistant MCP tooling so Codex can use structured
Home Assistant tools without requiring a separate MCP add-on installation.

See `DOCS.md` for setup, security, and operating details.
